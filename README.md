## Overview

This project is a professional-grade Trading Operations analytics dashboard built in Microsoft Excel.

It demonstrates a full end-to-end Business Intelligence pipeline using:

- Power Query (ETL Layer)
- Power Pivot (Data Model)
- DAX Measures (Analytics Engine)
- Pivot Table Controller (Filtering Layer)
- Cube Functions (Reporting Layer)
- Excel Dashboard Visualizations

The goal is to simulate an institutional trading operations reporting system.

# 🧠 Data Pipeline Architecture

```text
Raw Trade Data
    ↓
Power Query Cleaning (ETL)
    ↓
Power Pivot Data Model
    ↓
DAX Measures (Analytics Engine)
    ↓
Pivot Table Controller (Filters)
    ↓
Cube Formulas (Reporting Layer)
    ↓
Dashboard Visualizations
```
